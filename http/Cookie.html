class http\Cookie

A class representing a list of cookies with specific attributes.

Constants:#

  • PARSE_RAW
    Do not decode cookie contents.
  • SECURE
    The cookies' flags have the secure attribute set.
  • HTTPONLY
    The cookies' flags have the httpOnly attribute set.

Properties:#

None.
