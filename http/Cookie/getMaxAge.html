int http\Cookie::getMaxAge()

Get the currently set max-age attribute of the cookie list. See http\Cookie::setMaxAge().

NOTE:
A return value of -1 means that the attribute is not set.

Params:#

None.

Returns:#

  • int, the currently set max-age.
